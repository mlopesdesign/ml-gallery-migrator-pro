# ML Gallery Migrator Pro v1.0.26 – Stable Production Release

ML Gallery Migrator Pro v1.0.26 is the first production-ready public distribution build.

## Highlights
- **Security Hardening**: Implemented robust `manage_options` capability checks and `mlgmp_ajax` nonce verification.
- **Data Hygiene**: Added `uninstall.php` to ensure clean removal of database tables and options.
- **Stable Migration Engine**: Validated two-pass album mapping, batch processing, and shortcode conversion.
- **Multilingual**: Full support for Portuguese (Brazil), English (US), and Spanish (Spain).
- **Responsive UI**: Optimized for all admin screens, including mobile and split-view.

## Instructions
1. Download the attached `ml-gallery-migrator-pro-v1.0.26.zip`.
2. Install via WordPress Admin (Plugins > Add New > Upload).
3. Access the **ML Migrator NGG** menu.
